如果声明了 private字段，但在程序中未使用该字段，则可以将其视为死代码，因此应该删除该字段。 这将提高可维护性，因为开发人员不会想知道变量是用来做什么的。

不合规的代码示例

class MyClass {
  private $foo = 4;                       //foo 未使用

  public function compute($a) {
    return $a * 4;
  }
}

合规的解决方案

class MyClass {

  public function compute($a) {
    return $a * 4;
  }
}
