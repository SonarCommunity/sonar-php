未执行的private 方法是死码：即不必要的，无效的应该被删除的代码。清除死代码会减少维护的代码库的大小， 使程序更容易理解，并防止引入错误。

不合规的代码示例

public class Foo
{
  private function Foo() {}   // 合规，私有的空的构造函数故意用于防止任何类的直接实例化。

  public static function doSomething()
  {
    $foo = new Foo();
    ...
  }

  private function unusedPrivateFunction() {  // 不合规
  }
}

合规的解决方案

public class Foo
{
  private function Foo(){}   // 合规，私有的空的构造函数故意用于防止任何类的直接实例化。

  public static function doSomething()
  {
    $foo = new Foo();
  }
}
