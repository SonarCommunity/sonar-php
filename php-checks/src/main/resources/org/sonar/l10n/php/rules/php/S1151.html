switch语句应该只用于清楚地定义控制流中的一些新分支。只要一个case 子句包含太多语句，就会大大降低整个控制流语句的可读性。在这种情况下，应该将case 子句中的内容提取到单独的的方法中

不合规的代码示例

阈值为5时：

switch ($var) {
  case 0:  // 6行到下一个case
    methodCall1();
    methodCall2();
    methodCall3();
    methodCall4();
    break;
  default:
    break;
}

合规的解决方案

switch ($var) {
  case 0:
    doSomething();
    break;
  default:
    break;
}

function doSomething(){
  methodCall1("");
  methodCall2("");
  methodCall3("");
  methodCall4("");
}
