重复的字符串文本使重构过程容易出错，因为必须确保更新所有发生的情况。

另一方面，常量可以从许多地方引用，但只需要在一个地方进行更新。

不合规的代码示例

默认阈值为3时：

function run() {
  prepare('action1');                              // 不合规 - 'action1'被复制了3次
  execute('action1');
  release('action1');
}

合规的解决方案

ACTION_1 = 'action1';

function run() {
  prepare(ACTION_1);
  execute(ACTION_1);
  release(ACTION_1);
}

例外

为了防止产生一些误报，排除了少于5个字符的文字。
