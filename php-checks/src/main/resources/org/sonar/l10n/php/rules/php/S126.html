只要 if 语句后面跟一个或多个 else if 语句，这条规则就有用；最后的 else if 后面应该跟一个 else 语句。

最终要求的 else 语句是防御性编程。

else 语句应该采取适当的操作，或者包含关于为什么不采取操作的适当注释。这与在switch语句中最后使用default子句的要求是一致的。

不合规的代码示例

if (condition1) {
  do_something();
} elseif (condition2) {
  do_something_else();
}

合规的解决方案

if (condition1) {
  do_something();
} elseif (condition2) {
  do_something_else();
} else {
  throw new InvalidArgumentException('message');
}
