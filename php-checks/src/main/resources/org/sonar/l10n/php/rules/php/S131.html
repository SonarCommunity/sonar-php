最后的case default 语句是防御性编程需求。该语句要么采取适当的行动，要么包含关于为什么不采取行动的适当说明。 即时 switch 覆盖了enum的所有当前值, default case任然会被使用，因为不能保证enum 不会被扩展。

不合规的代码示例

switch ($param) {  //缺少default语句
  case 0:
    do_something();
    break;
  case 1:
    do_something_else();
    break;
}

合规的解决方案

switch ($param) {
  case 0:
    do_something();
    break;
  case 1:
    do_something_else();
    break;
  default:
    error();
    break;
}
