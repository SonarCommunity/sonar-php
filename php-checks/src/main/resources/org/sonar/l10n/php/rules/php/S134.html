嵌套的if，for，while，switch，和try 语句是制作所谓“意大利面代码”的关键成分。

这样的代码很难读取、重构和维护。

不合规的代码示例

当默认阈值为3时:

  if (condition1) {                  // 合格 - depth = 1
    ...
    if (condition2) {                // 合格 - depth = 2
      ...
      for($ = 0; $i < 10; $i++) {  // 合格 - depth = 3,不超过限额
        ...
        if (condition4) {            // 不合格 - depth = 4
          if (condition5) {          // Depth = 5, 超出限制，但只有 depth = 4时才报告问题
            ...
          }
          return;
        }
      }
    }
  }
