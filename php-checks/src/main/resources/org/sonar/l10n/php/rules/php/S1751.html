最多有一个迭代的循环相当于使用 if语句有条件地执行一段代码。没有开发人员希望找到循环语句的这种用法。 如果作者的初始意图是有条件地执行一个代码，则应改为使用if语句。

在最坏的情况下，这并不是作者最初的意图，所以循环体应该被修复，应该采用嵌套的 return, break 或者 throw 语句为更合适的方式。

不合规的代码示例

for ($i = 0; $i < 10; $i++) { // 不合规
  echo "i is $i";
  break;
}
...
for ($i = 0; $i < 10; $i++) { // 不合规
  if ($i == $x) {
    break;
  } else {
    echo "i is $i";
    return;
  }
}

合规的解决方案

for ($i = 0; $i < 10; $i++) {
  echo "i is $i";
}
...
for ($i = 0; $i < 10; $i++) {
  if ($i == $x) {
    break;
  } else {
    echo "i is $i";
  }
}
