跳转语句(return，break，continue， goto) 和throw 表达式将控制流移出当前代码块。所以在跳转之后出现的任何未标记的语句都是死码。

不合规的代码示例

function fun($a) {
  $i = 10;
  return $i + $a;
  $i++;             // 死码
}

合规的解决方案

function fun($a) {
  $i = 10;
  return $i + $a;
}
