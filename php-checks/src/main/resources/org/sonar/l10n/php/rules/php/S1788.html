为方法参数定义默认值的能力可以使方法更容易使用。默认参数值允许调用者指定任意多或少的参数，同时获得相同的功能和最小化样板文件、包装程序代码。

但是，所有带有默认值的方法参数都应该在没有默认值的方法参数之后声明。否则，调用者就不可能利用默认值;它们必须重新指定默认值，以便“获得”非默认参数。

不合规的代码示例

function makeyogurt($type = "acidophilus", $flavor){...}  // 不合规

makeyogurt("raspberry")}}  // 运行时错误:调用makeyogurt()时缺少参数2

合规的解决方案

function makeyogurt($flavor, $type = "acidophilus", ){...}

makeyogurt("raspberry")}} // 像预期的那样工作
