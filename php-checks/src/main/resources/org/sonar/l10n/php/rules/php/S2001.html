不推荐使用的语言功能是为了向后版本兼容而临时保留的功能，但最终都将从语言中删除。 实际上，弃用宣布了一个宽限期，允许从旧功能到新功能的平稳过渡。 在此期间，不应该在代码中添加任何已废弃特性的使用，并且应该逐步删除所有现有的使用。

PHP5不支持以下函数:

不支持 更新代替
call_user_method() call_user_func()
call_user_method_array() call_user_func_array()
define_syslog_variables()
dl()
ereg() preg_match()
ereg_replace() preg_replace() (注意，这在PHP 5.5中是不推荐的)
eregi() preg_match() 使用'i'修饰符
eregi_replace() preg_replace() 使用'i'修饰符
set_magic_quotes_runtime() 和它的别名,magic_quotes_runtime()
session_register() $_SESSION superglobal
session_unregister() $_SESSION superglobal
session_is_registered() $_SESSION superglobal
set_socket_blocking() stream_set_blocking()
split() preg_split()
spliti() preg_split() 使用'i'修饰符
sql_regcase()
mysql_db_query() mysql_select_db() 和 mysql_query()
mysql_escape_string() mysql_real_escape_string()
将区域设置类别名称作为字符串传递 使用LC_*常量族

不支持 更新代替
__autoload() spl_autoload_register()
create_function() 匿名函数
parse_str() 没有第二个参数 parse_str() 伴随第二个参数
gmp_random() gmp_random_bits() 或 gmp_random_range()
each() foreach
assert() 伴随字符串参数
通过调用define()定义不区分大小写的常量，并将true 作为第三个参数 define("myconst", $value) 或 define("myconst", $value, false)
FILTER_FLAG_SCHEME_REQUIRED 和 FILTER_FLAG_HOST_REQUIRED 标志 FILTER_VALIDATE_URL 标志
fgetss() 函数，"string.strip_tags" 流过滤的名字, SplFileObject::fgetss() 方法和 gzgetss() 函数
mbregex_encoding(), mbereg(), mberegi(), mbereg_replace(), mberegi_replace(), mbsplit(), mbereg_match(), mbereg_search(), mbereg_search_pos(), mbereg_search_regs(), mbereg_search_init(), mbereg_search_getregs(), mbereg_search_getpos(), mbereg_search_setpos() 使用
相应的 mb_ereg_*() 变体代替
带整数指针的字符串搜索函数(stristr, strrchr, strstr, strripos, stripos, strrpos, strpos, strchr) 用字符串指针代替
image2wbmp() imagewbmp()
Normalizer::NONE
在名称空间中定义一个assert()函数 使用标准的assert()函数
