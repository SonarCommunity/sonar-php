PHP有两组逻辑运算符：&& / || 和and / or。 这两组之间的区别是优先权。 因为and / or 优先级几乎低于任何其他操作符，使用它们代替&& / ||可能不会得到期望的结果。

不合规的代码示例

$have_time = true;
$have_money = false;
$take_vacation = $have_time and $have_money;  // 不合规。 $take_vacation == true.

合规的解决方案

$have_time = true;
$have_money = false;
$take_vacation = $have_time && $have_money;  // $take_vacation == false.
