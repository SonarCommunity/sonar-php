$this 指的是当前类实例。但是静态方法可以在不实例化类的情况下访问，并且 $this 对它们不可用。在静态上下文中使用 $this 将在运行时导致致命错误。

不合规的代码示例

class Clazz {
  $name=NULL;  // 实例变量

  public static function foo(){
    if ($this->name != NULL) {
      // ...
    }
  }
}

合规的解决方案

class Clazz {
  $name=NULL;  // 实例变量

  public static function foo($nameParam){
    if ($nameParam != NULL) {
      // ...
    }
  }
}
