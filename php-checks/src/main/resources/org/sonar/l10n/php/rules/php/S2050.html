PHP中的某些函数仅作为其他函数的别名存在。这些别名是为了向后版本兼容而提供的，但实际上应该从代码中删除。

这条规则寻找以下别名的用法：

别名 更换
chop rtrim
close closedir
doubleval floatval
fputs fwrite
ini_alter ini_set
is_double is_float
is_integer is_int
is_long is_int
is_real is_float
is_writeable is_writable
join implode
key_exists array_key_exists
magic_quotes_runtime set_magic_quotes_runtime
pos current
show_source highlight_file
sizeof count
strchr strstr

不合规的代码示例

$arr=array("apple", "pear","banana");
echo sizeof($arr);  // 不合规

合规的解决方案

$arr=array("apple", "pear","banana");
echo count($arr);
