当依靠密码认证模式进行数据库连接时，应选择一个安全密码。

使用空密码时，此规则会引起问题。

不合规的代码示例

// 连接到mysql数据库时空密码的示例
$conn = new mysqli($servername, $username, "");

合规的代码示例

// 生成安全密码，将其设置为用户名数据库，并将其存储在例如环境变量中
$password = getenv('MYSQL_SECURE_PASSWORD');
// 然后连接到数据库
$conn = new mysqli($servername, $username, $password);
