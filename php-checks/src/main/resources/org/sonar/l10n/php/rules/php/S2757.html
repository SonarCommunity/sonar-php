使用运算符对( =+，=- 或者 =! )进行反向运算时，单个运算符意味着(+=， -= 或者 !=) 将编译并运行，但不会产生预期的结果。

当使用 =+，=- 或者 =! 时，这两个操作符之间没有任何间距，并且后面至少有一个空格字符时，这条规则会引发一个问题。

不合规的代码示例

$target = -5;
$num = 3;

$target =- $num;  // 不合规；target = -3。这是什么意思？
$target =+ $num; // 不合规；target = 3

合规的解决方案

$target = -5;
$num = 3;

$target = -$num;  // 合规；分配num逆值的意图很明显
$target += $num;
