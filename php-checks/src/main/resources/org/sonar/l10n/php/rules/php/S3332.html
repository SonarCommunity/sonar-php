没有固定生存期或失效日期的cookie称为非持久性cookie，或“会话（session）”cookie，这意味着它们只在浏览器会话期间有效，而当浏览器关闭时，它们就会消失。具有失效日期的Cookies，即"持久（persistent）" cookies将被存储/保存到这些日期。

非持久性cookie应用于管理网站上的登录会话。要使cookie非持久化，只需省略 expires属性。

当会话cookie设置了expires 时，该规则会引发一个问题，这个可以通过编程方式设置，也可以通过配置设置，比如session.cookie_lifetime。
