enable_dl 默认是打开的，并且允许忽略open_basedir 限制，这限制脚本可以访问的文件。因此，这是一个危险的选择，应该明确地关闭。

当enable_dl 在 php.ini中没有显式设置为0时，该规则会引发一个问题。

不合规的代码示例

; 在php.ini中
enable_dl=1  ; 不合规

合规的解决方案

; 在php.ini中
enable_dl=0
