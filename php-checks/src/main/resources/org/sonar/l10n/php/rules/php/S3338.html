file_uploads 是默认打开的PHP配置，允许将文件上载到站点。因为从陌生人那里接受 candy文件本身就是危险的，除非对站点绝对必要，否则应该禁用该特性。

当 file_uploads 未显式禁用时，此规则会引发一个问题。

不合规的代码示例

; 在php.ini中
file_uploads=1  ; 不合规

合规的解决方案

; 在php.ini中
file_uploads=0
