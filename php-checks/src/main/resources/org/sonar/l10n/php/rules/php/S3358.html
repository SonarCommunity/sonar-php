仅仅因为你 can 可以做一些事，并不意味着您应该这样做，这就是嵌套三元操作的情况。嵌套三元运算符导致这类代码在编写时看起来很清晰，但是六个月后，维护者(或者更糟的是，或许就是你)会感到困惑和诅咒。

相反，为了清晰起见，应该使用另一行来将嵌套操作表示为单独的语句。

不合规的代码示例

function get_title($gender, $is_married) {
  return $gender == "MALE" ? "Mr. " : ($is_married ? "Mrs. " : "Miss ");  // 不合规
}

合规的解决方案

function get_title($gender, $is_married) {
  if ($gender == "MALE") {
    return "Mr. ";
  }
  return $is_married ? "Mrs. " : "Miss ";
}
