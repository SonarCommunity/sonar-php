因为PHP是动态类型的，所以PHP不会对函数强制执行返回类型。这意味着函数中的不同路径可以返回不同类型的值，这可能会让用户非常困惑，而且维护起来也非常困难。

特别是，因此也可以将空的 return 语句 (隐式返回 null) 与一些返回表达式混合使用。此规则验证函数中的所有 return 语句是否一致。

不合规的代码示例

function foo($a) { // 不合规，函数将返回"true"或null
  if ($a == 1) {
    return true;
  }
  return;
}

合规的解决方案

function foo($a) {
  if ($a == 1) {
    return true;
  }
  return false;
}

或者

function foo($a) {
  if ($a == 1) {
    return true;
  }
  return null;
}
