将所有分支放在 switch 或者 if 链中并使用相同的实现是错误的。要么复制粘贴错误，应该执行一些不同的操作，要么根本不应该有一个 switch/if 链。

不合规的代码示例

if ($b == 0) {  // 不合规
  doOneMoreThing();
} else {
  doOneMoreThing();
}

$b = $a > 12 ? 4 : 4;  // 不合规

switch ($i) {  // 不合规
  case 1:
    doSomething();
    break;
  case 2:
    doSomething();
    break;
  case 3:
    doSomething();
    break;
  default:
    doSomething();
}

例外

此规则不适用于 if 链没有 else-s，或者switch-es 没有 default子句。

if($b == 0) {    //没有问题，这样做的目的是使代码更具可读性
  doSomething();
} elseif($b == 1) {
  doSomething();
}
