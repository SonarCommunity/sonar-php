当每个语句都有自己的行时，代码最清楚。尽管如此，将if 及其结果 then 语句组合在同一行是一种常见的模式。但是，当if 于前面的 else 或 elseif 的结束代码 } 放在同一行时， 它要么是个错误——没有else，要么在维护者不理解这两个语句是不连接的情况下，导致将来出现错误。

不合规的代码示例

if ($condition1) {
  //......
} if ($condition2) {  // 不合规
  //......
}

合规的解决方案

if ($condition1) {
  //......
} elseif ($condition2) {
  //......
}

或者

if ($condition1) {
  //......
}

if ($condition2) {
  //......
}
