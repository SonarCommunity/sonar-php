数组或可计数对象中的元素个数始终大于或等于零。因此，测试count是否大于或等于零是没有意义的，因为结果总是 true。类似地，测试它是否小于零总是返回false。也许其目的是检查对象或数组的非空性。

不合规的代码示例

if (count($arr) >= 0) { ... }

if (count($arr) < 0) { ... }

$result = count($arr) >= 0;

if (0 > count($arr)) { ... }

合规的解决方案

if (count($arr) != 0) { ... }

if (count($arr) > 0) { ... }
