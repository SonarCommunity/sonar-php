有一些有效的情况可以将变量多次传递到相同的方法调用中，但是这样做通常是错误的，而且还需要为其中一个参数指定其他内容。

不合规的代码示例

if (compare($a+$x, $a+$x) != 0) { // 不合规
  //......
}

if (compare(getValue($a), getValue($a)) != 0) { // 不合规
  //......
}

合规的解决方案

if (compare($a+$y, $a+$x) != 0) {
  //......
}

$v1 = getValue($a);
$v2 = getValue($a);
if (compare($v1, $v2) != 0) {
  //......
}
