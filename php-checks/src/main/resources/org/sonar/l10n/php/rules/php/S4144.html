当两个方法具有相同的实现时，要么是一个错误—另有目的—要么是有意进行的复制，但可能会让维护人员感到困惑。在后一种情况下，一个实现应该调用另一个实现。

不合规的代码示例

class A {
    private const CODE = "bounteous";

    public function getCode() {
        doTheThing();
        return A::CODE;
    }

    public function getName() {  // 不合规
        doTheThing();
        return A::CODE;
    }
}

合规的解决方案

class A {
    private const CODE = "bounteous";

    public function getCode() {
        doTheThing();
        return A::CODE;
    }

    public function getName() {
        return $this->getCode();
    }
}

例外

忽略不是访问器（getter和setter）且语句数少于2条的方法。
