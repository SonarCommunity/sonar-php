事实证明，较旧的SSL / TLS协议（例如“ SSLv3”）是不安全的。

当使用不安全的协议版本（即：不同于“ TLSv1.2”，“ TLSv1.3”，“ DTLSv1.2”或“ DTLSv1.3”的协议）创建SSL / TLS上下文时，会引起问题。

不合规的示例

$ctx = stream_context_create([
  'ssl' => [
    'crypto_method' =>
      STREAM_CRYPTO_METHOD_TLSv1_1_CLIENT // 不合规
  ],
]);

合规的方案

$ctx = stream_context_create([
    'ssl' => [
        'crypto_method' => STREAM_CRYPTO_METHOD_TLSv1_2_CLIENT
    ],
]);
