并不是所有的SSL协议都是平等创建的，一些像“SSL”这样的遗留协议已经被证明是不安全的。

当使用不安全协议(即不同于“TLSv1.2”或“DTLSv1.2”的协议)创建SSL上下文时，此规则会引发一个问题。

不合规的代码示例

$ctx = stream_context_create([
  'ssl' => [
    'crypto_method' =>
      STREAM_CRYPTO_METHOD_TLSv1_1_CLIENT // 不合规
  ],
]);

合规的解决方案

$ctx = stream_context_create([
    'ssl' => [
        'crypto_method' => STREAM_CRYPTO_METHOD_TLSv1_2_CLIENT
    ],
]);
