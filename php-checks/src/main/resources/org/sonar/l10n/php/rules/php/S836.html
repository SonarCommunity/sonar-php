在PHP中，不需要在使用变量之前初始化变量。 但是，使用未初始化的变量被认为是不好的做法，由于以下原因，应避免使用它们：

  • 未初始化变量的值和类型取决于其首次使用的上下文。 最好对这些内容进行明确说明，以免造成混淆。
  • 在许多情况下，解释器会抛出warning或notice。

不合规示例

<?php

function getText(array $lines): string {
    foreach ($lines as $line) {
        $text .= $line;
    }

    return $text;
}

合规示例

<?php

function getText(array $lines): string {
    $text = "";

    foreach ($lines as $line) {
        $text .= $line;
    }

    return $text;
}
