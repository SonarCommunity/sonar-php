goto 是一种非结构化的控制流语句。它降低了代码的可读性和可维护性。应替换为结构化的控制流语句 如 if， for， while， continue 或是 break 这些语句。

不合规的代码示例

$i = 0;
loop:
  echo("i = $i");
  $i++;
  if ($i < 10){
    goto loop;
  }

合规的解决方案

for ($i = 0; $i < 10; $i++){
  echo("i = $i");
}
